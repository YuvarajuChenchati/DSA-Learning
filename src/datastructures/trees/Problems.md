## üí° Problem-Solving Strategies

### Tree Traversal Patterns
- **DFS (Pre/In/Post)**: Recursive or iterative with stack
- **BFS (Level Order)**: Queue-based traversal
- **Morris Traversal**: O(1) space for inorder

### Common Approaches
1. **Recursive**: Natural for tree problems
2. **Iterative**: Use stack for DFS, queue for BFS
3. **Two Passes**: First pass for info, second for solution
4. **Hash Map**: Store node information
5. **Parent Pointers**: Track parent nodes

### BST Specific
- **Inorder Traversal**: Returns sorted order
- **BST Property**: Left < Root < Right
- **Range Checking**: Validate bounds
- **Successor/Predecessor**: Find next/previous node
